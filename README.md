# Spreadsheet Uploader Tool


To run: 

# Backend: 

from backend-app: 
``
```
python -m venv env
source env/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
docs available with SwaggerUI at /docs
requirements in requirements.txt

# Frontend
from frontend-app: 
```
npm install
npm run dev
```
